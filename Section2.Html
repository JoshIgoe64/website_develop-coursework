2.1.7 Describe which are the appropriate file types for websites
HTML is the standard for the actual website file(HyperText Markup Language), and GIF, PNG and JPEG are the common file used to complement this in images gifs and videos. PNG is the most common file type on a website as this was designed specifically for websites. Javascript and Perl are also commonly used as scripts on a website to complement the HTML and CSS. I will be using HTML for the base, CSS for style and Javascript for backend
